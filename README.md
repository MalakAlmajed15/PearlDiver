# 🎮 Pearl Diver – Bahrain’s Treasure

## 📌 Overview
A 3D underwater adventure game developed in Unity (URP), inspired by Bahrain’s pearling heritage.

## 🚀 Features
- Main Menu (Start, Instructions, Settings, Credits, Quit)
- 3D underwater exploration
- Pearl collection system
- Air meter (breath system)
- Lives system (3 hearts)
- Hostile sea creatures
- Time-based medals & scoring
- Multiple levels

## 🛠️ Technologies
- Unity 6.3 (URP)
- C#

## 🎯 How to Play
- WASD → Move  
- Mouse → Look  
- Space / Shift → Up / Down  
- Collect pearls and avoid enemies
